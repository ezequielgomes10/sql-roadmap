# SQL Roadmap - Fundamentos para Iniciantes

> Roadmap interativo e visual para quem está começando com SQL. Aprenda os fundamentos na prática, com exemplos reais de negócio.

🔗 **[Acesse o projeto ao vivo aqui](https://ezequielgomes10.github.io/sql-roadmap/)** 

---

## 📌 Sobre o projeto

Este projeto nasceu do meu próprio processo de aprendizado em SQL. O objetivo é oferecer um guia visual, interativo e direto ao ponto para iniciantes que querem entender SQL de verdade — não apenas decorar comandos.

O roadmap cobre desde os primeiros passos até JOINs, com estudos de caso que simulam situações reais de empresas.

---

## 🗂️ Conteúdo

| Módulo | Temas abordados |
|--------|----------------|
| 01 · Seleção e Filtros | SELECT, FROM, WHERE, LIKE, IN, BETWEEN, NULL |
| 02 · Ordenação e Agrupamento | ORDER BY, GROUP BY, HAVING |
| 03 · Agregações | COUNT, SUM, AVG, MIN, MAX |
| 04 · JOINs | INNER JOIN, LEFT JOIN, relações entre tabelas |

**Além disso:**
- 5 estudos de caso com dados reais de negócio (CRM, vendas, financeiro, operações)
- Seção sobre SGBDs: PostgreSQL, MySQL, SQLite e SQL Server
- Boas práticas e erros comuns de iniciantes
- Rastreamento de progresso interativo
